# Double M — Portfolio V2

Site vitrine / portfolio premium pour Double M.

## Structure

- `index.html` : accueil
- `projets.html` : page toutes les études de cas
- `projets/*.html` : pages projets détaillées
- `style.css` : design responsive
- `script.js` : menu mobile + animations
- `assets/images` : logos, visuels et mockups

## Déploiement Vercel

Si le dossier est à la racine du repo, ne renseignez pas de Root Directory. Si le dossier complet est uploadé dans le repo, indiquez :

```txt
double-m-portfolio-v2
```

ou renommez le dossier en `double-m-portfolio` selon votre upload.
