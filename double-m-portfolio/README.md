## Mise en ligne GitHub

```bash
git clone https://github.com/matteomaestracci131/Double-M.git
cd Double-M
# Copier les fichiers du dossier double-m-portfolio ici
git add .
git commit -m "Ajout V1 site Double M"
git push origin main
```

## Déploiement Vercel

1. Aller sur Vercel.
2. Importer le repository GitHub `Double-M`.
3. Garder les paramètres par défaut.
4. Cliquer sur Deploy.

## À personnaliser ensuite

- Remplacer le lien LinkedIn dans `index.html`.
- Ajouter les visuels réels des projets.
- Ajouter des pages projet détaillées.
- Ajouter les témoignages clients.
